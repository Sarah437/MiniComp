
# MiniComp

Projeto acadêmico da disciplina de Compiladores.

## Objetivo

Implementar um compilador simplificado capaz de:

- Realizar análise léxica
- Realizar análise sintática
- Gerar código intermediário

## Estrutura

```txt
src/
docs/
exemplos/
```

## Execução

```bash
cd src

python main.py
```

## Exemplo

Entrada:

```txt
x = 10
y = 20
```

Saída:

```txt
LOAD 10
STORE x

LOAD 20
STORE y
```
