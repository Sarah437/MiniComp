
# Estudo de Caso

Compilador analisado: GCC

O GCC (GNU Compiler Collection) é um dos compiladores mais utilizados no mundo.

Características:

- Código aberto
- Multiplataforma
- Suporte a diversas linguagens
- Alto desempenho

O fluxo de compilação inclui:

1. Análise Léxica
2. Análise Sintática
3. Análise Semântica
4. Otimização
5. Geração de Código
