
# Relatório Final

## Objetivo

Desenvolver um compilador simplificado capaz de
reconhecer atribuições e gerar código intermediário.

## Tecnologias

- Python 3
- Git
- GitHub

## Funcionalidades

- Tokenização
- Validação sintática
- Geração de código

## Exemplo

Entrada:

x = 10

Saída:

LOAD 10
STORE x

## Conclusão

O projeto permitiu aplicar conceitos fundamentais
de compiladores, especialmente análise léxica,
análise sintática e geração de código.
